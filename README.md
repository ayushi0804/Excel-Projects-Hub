📊 Sales Performance Dashboard (Microsoft Excel)

📌 Project Overview

This is a beginner-friendly Excel dashboard project that analyzes sales performance using Pivot Tables, Pivot Charts, Slicers, and VBA automation. The dashboard provides interactive insights into employee sales, regional performance, target achievement, and sales trends.

🚀 Features

* Interactive Sales Performance Dashboard
* Dynamic Pivot Tables and Pivot Charts
* Region-based filtering using Slicers
* VBA Macro for dynamic slicer connections
* Total Sales and Target Performance Analysis
* Employee-wise Sales Comparison
* Target Hit % and Away From Target % Analysis
* Clean and user-friendly dashboard layout

🛠️ Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* VBA (Visual Basic for Applications)
* Excel Formulas

📈 Dashboard Insights

* Analyze employee sales performance.
* Compare target achievement across employees.
* Filter reports by region using interactive slicers.
* Visualize sales trends through charts.
* Monitor Target Hit % and Away From Target %.
